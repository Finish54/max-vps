# 🎨 Дизайн-референс: vlessi.me

**Цель:** использовать как основу для лендинга MAX VPS на Flutter Web.
**Источник:** https://vlessi.me (парсинг 20.06.2026)
**Файлы:** HTML 3.8 КБ, CSS 131 КБ, JS bundle 2.9 МБ (SPA)

---

## 🏗 Архитектура сайта

**vlessi.me — это SPA** (Single Page Application):
- HTML почти пустой (3.8 КБ) — только `<div id="app">` и meta-теги
- Всё рендерится через JavaScript (2.9 МБ bundle)
- Используется **Vue или React** (стиль похож на Nuxt/Next)
- Серверная часть (SSR) — **нет** (это подтверждает почему vlessi.me так трудно парсить)

> **Для MAX VPS:** Это значит что Flutter Web SPA **похож по подходу** — будем делать SEO-зеркало отдельно.

```html
<meta name="keywords" content="vpn, vpn подписка, vpn для телефона, vpn для роутера, vless, vpn россия, домашний vpn, мобильный vpn, купить vpn, влессиvpn, vlessi" />

```

### Open Graph

```html
<meta property="og:type" content="website" />
<meta property="og:title" content="ВлессиVPN — VPN-подписки для дома и телефона" />
<meta property="og:url" content="https://vlessi.me/" />
<meta property="og:site_name" content="ВлессиVPN" />
```

> **⚠️ Но:** у vlessi.me **нет** `<meta property="og:image">` в head (плохо для шеринга!)
> **Наша задача:** добавить og:image, twitter:card, schema.org

## 🎯 Что мы делаем в MAX VPS (отличия от vlessi.me)

### Берём

✅ **Палитру** (тёмная тема `#05080f` + синий акцент `#3080ff`)
✅ **Шрифты** (Unbounded + Inter)
✅ **Структуру блоков** (hero → преимущества → тарифы → FAQ → footer)
✅ **Подход к адаптиву**
✅ **Юридические страницы** (та же структура)

### Делаем лучше

🎨 **Логотип** — наш SVG (щит + текст MAX VPS)
🎨 **Галерея** — больше визуала (скриншоты приложения)
🎨 **Live-демо** — попробовать без регистрации
🎨 **SEO** — og:image 1200×630, twitter:card, JSON-LD schema.org
🎨 **Telegram Mini App** — вход по кнопке в боте (без логина/пароля)
🎨 **Flutter Web** — плавные анимации (Hero, Tween)

### НЕ берём

❌ Login/password (используем Telegram-код)
❌ Реферальная программа в таком виде (сделаем проще)
❌ Большой JS bundle (Flutter Web эффективнее)
